NE555 - Operation_Modes

Aufgabenstellung

  • Simuliere die Schaltung ‘NE555 in Astabiler Operation’, mit Output Frequenz von 10kHz.
  • Erstelle eine Schaltung, welche ein PWM Signal generiert, die mittels einer analogen Spannung (Wechselspannung) eingestellt werden kann.

Grundschaltung Astabel Operation

Ra = R2

Rb = R1

C = C1

Berechnung

  • Einzustellende Frequenz: 10kHz
  • Widerstandswert Ra = 50kOhm (angenommen)
  • Widerstandswert Rb = 25kOhm
    • Ergibt sich aus ⇒ Ra+2Rb=100KOhmRa + 2Rb=100KOhmRa+2Rb=100KOhm﻿
  • Kondensatorwert C = 0,01uF (abgelesen)
  • Berechnet C: frequency=1.44/(RA+2RB)C frequency = 1.44/(RA + 2RB)C frequency=1.44/(RA+2RB)C ﻿
    • 10k=1,44/(50k+2∗25k)∗C=1,44nF10k = 1,44/(50k+2*25k)*C = 1,44nF10k=1,44/(50k+2∗25k)∗C=1,44nF﻿

Blau → Stellt die am Kondensator anliegende Spannung dar.

Grün → Stellt die Ausgangsspannung an Pin Out dar.

Auswertung

R2 in Kombination mit R1 ergibt einen Spannungsteiler, mit welchem die Ausgangsfrequenz eingestellt werden kann.

Zum Zeitpunkt t = 0 ist der Kondensator C nicht geladen.

Mittels des in den NE555 integrierten Spannungsteiler (R=R=R) wird das FF bezüglich der außen anliegenden Spannung an TRIG & THRES gesetzt.

Am Kopf des am nähersten an GND befindlichen Widerstandes beträgt die Spannung immer 13∗Vcc\frac{1}{3}*V_{cc} 31​∗Vcc​﻿.

Über das Widerstandsverhältnis R1&R2 lädt sich der externe Kondensator in einer über das Widerstandsverhältnis definierten Zeit bis auf die Spannung 23∗Vcc\frac{2}{3}*V_{cc} 32​∗Vcc​﻿ auf, in dieser Zeit ist das FF zwecks des negierten Eingang am Comp2 auf Rücksetzen gesetzt → Out gibt, weil es negiert ist 1 aus.

Sobald die Threshhold-Spannung von 23∗Vcc\frac{2}{3}*V_{cc} 32​∗Vcc​﻿ erreicht wird, kippt die Schaltung, da gilt Trig kann Thres überschreiben. → Transistor wird niederohmig → Disch lieg auf GND → Externer Kondensator entlädt sich bis auf die Spannung 13∗Vcc\frac{1}{3}*V_{cc} 31​∗Vcc​﻿.

Grundschaltung Pulse Width Modulation

R2 = Rlast = 1k Ohm

R3 = Ra = 3k Ohm

C = C1= 0.02uF

Auswertung

Grün → Stellt die PWM Modulierte Ausgangsspannung an Pin OUT dar.

Blau → Stellt die Analoge sinusförmige Eingangsspannung an Pin CV dar.

Die Clock Input Voltage wird mittels eines NE555 in Monostabiler Operation bereitgestellt.

Wobei mithilfe der Frequenz der Clock Input Voltage die Sample Rate der Schaltung definiert werden kann.

[interne Schaltung] Die Spannung am Fuße des Widerstandes, welcher sich am nähersten an Vcc befindet, wird durch die Modulation Eingangs Spannung abgeändert.

Sobald der Trigger Impuls low ist, wird der interne DISC Transistor hochohmig und der externe Kondensator lädt sich auf. Der Kondensator lädt sich solange auf, bis diese Spannung gleich bzw. höher als die Modulation Input Spannung ist. Ist dies der Fall, wechselt das FF seinen Zustand und der DISC Transistor wird hochohmig, sodass C entlädt. (Auch ersichtlich in Abb. unten)

Blau → Trigger

Grün → Kondensator Spannung

Rot → Input Modulation Spannung
